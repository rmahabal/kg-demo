Banking Knowledge Graph
● Demo walkthrough · synthetic data · not production

The institution's knowledge, as a graph agents can ground on.

A pre-built knowledge graph for a credit union — products, pricing, rules, disclosures and the regulations that govern them — that agents reason over instead of guessing. Built from public sources, awaiting the institution's sign-off.

All four pieces below run on the real California Credit Union graph (445 knowledge nodes, built from public sources; no member data). It's a design prototype, not the production UI.
